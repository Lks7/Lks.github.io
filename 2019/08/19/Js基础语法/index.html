一、JavaScript基础

1 组成： ECMAScript BOM DOM
2 数据类型：
undefined 变量是未定义类型
number 变量是数值类型，不管整数还是小数
string 变量是字符串类型
boolean 变量是true或者false
object 变量是null类型，或者数组，对象，函数。
3 string类型常用的属性和方法
属性 .length 求长度
方法 .indexOf(str,index) 从index位置开始，找到str字符串第一次出现的位置
.subString(index1,index2) 截取下标再index1到index2位置的字符串，
.split(str) 根据str将字符串拆分为一个数组
4 数组的属性和方法
属性 .length 获取数组的长度
方法 .join(str) 将数组用str连接符连成一个字符串
.sort() 将数组排序
.push(str) 将str元素添加到数组的最后，并返回新的数组长度
5 常用系统函数
.parseInt(“”); 把字符串转换成整数
parseFloat(“”); 把字符串转换成浮点数
isNaN(a); 判断a是不是非数字，若是非数字则返回true,若是数字则返回false
6 常见的事件
onload() 加载页面时调用
onclick() 鼠标的单击事件
onmouseover() 鼠标移到某元素之上
onkeydown() 键盘被按下
onchange() 域的内容被改变

二、JavaScript操作BOM对象

1.window对象常用的方法
window.alert(“”); 弹出对话框，有提示信息和一个确定按钮
window.prompt(“”,””); 用户可输入的对话框
window.confirm(“”); 弹出对话框，有提示信息和一个确定按钮，一个取消按钮
点击确定按钮返回true,点击取消按钮返回false
window.setTimeout() 定时调用函数(1次)
window.setInterval() 定时调用函数(按指定周期)
2.history对象的常用方法
history.go() 跳转到某个具体页面
history.go(-1) = history.back() 返回到上一个页面
history.go(1) = history.forward() 前进到下一个页面
3.location对象的常用方法
location.href 获取当前页面的url
4.document对象的常用方法
属性：document.referrer 返回载入当前页面的url
document.URL 返回当前页面的url
方法：document.getElementById(“id名”) 通过id获取元素
document.getElementsByName(“name名”) 过name属性值获取元素，返回值为集合
document.getElementsByTagName(“标签名”) 通过标签名获取元素，返回值为集合
document.write() 向页面输出文字

innerHTML     可以设置或获取段落或标题上面的文字部分
value         可以设置或获取按钮上面的文字

5.Date对象获取时间和日期
var 名=new Date(); 首先创建Date对象
名.getFullYear() 获取当前的年份
名.getMonth() 获取当前月是一年中的第几月 0-11
名.getDate() 获取一个月的第几天 1-31
名.getDay() 获取当前的日期是一年周中的第几天 0-6
6.Math对象
Math.random() 随机产生一个0-1之间的数，可得0，不能得1
数.toFixed(n) 四舍五入，参数n表示保留的小数位数
parseInt(Math.random()*(上限-下限+1)+下限); 生成一个(下限–上限)的随机数

三、JavaScript操作DOM对象

1 DOM操作分类 DOM Core HTML-DOM CSS-DOM
2 节点的层次关系 父节点（parent） 子节点（child） 兄弟节点（sibling）
3 使用层次关系访问节点
下一个节点 = oParent.nextElementSibling || oParent.nextSibling
前一个节点 = oParent.previousElementSibling || oParent.previousSibling
第一个子节点 = oParent. firstElementChild || oParent.firstChild
最后一个子节点 = oParent.lastElementChild || oParent.lastChild
4 获取节点信息
nodeName：节点名称
nodeValue：节点值
nodeType：节点类型
5 操作节点属性
getAttribute(“属性名”) 读取属性
setAttribute(“属性名”,”属性值”) 设置属性
6 增删改节点
createElement( tagName) 增加节点
A.appendChild( B) B追加到A的末尾
insertBefore( A,B ) A插入到B之前
cloneNode(deep) 复制节点
removeChild( node) 删除节点
replaceChild( newNode, oldNode) 替换节点
7 操作节点样式
HTML元素.style.样式属性＝”值” 设置样式
HTML元素.className=”样式名称” 设置class样式

四、JavaScript对象及初识面向对象

1 常见的内置对象
String（字符串）对象
Date（日期）对象
Array（数组）对象
Boolean（逻辑）对象
Math（算数）对象 使用时无需new
RegExp对象 （正则表达式）
2 JavaScript 中的对象分为普通对象和函数对象
普通对象类似于java中的实例对象，函数对象类似于java中的类，函数对象有构造函数
3 每个对象都有 proto 属性，但只有函数对象才有 prototype 属性
4 原型对象主要作用是用于继承。原型和原型链是JS实现继承的一种模型。
5 组合继承：有时也叫做伪经典继承
将原型链和借用构造函数的技术组合到一块，发挥二者之长的一种继承模式
使用原型链实现对原型属性和方法的继承，而通过借用构造函数来实现对实例属性的继承

五、初识jQuery

1 $(document).ready()与window.onload区别
window.onload 必须等待网页中所有的内容加载完毕后（包括图片、flash、视频等）才能执行
$(document).ready() 网页中所有DOM文档结构绘制完毕后即刻执行，
可能与DOM元素关联的内容（图片、flash、视频等）并没有加载完
window.onload 同一页面不能同时编写多个
$(document).ready() 同一页面能同时编写多个 简化写法 $(function(){ //执行代码});
2 jQuery代码风格
“$”等同于“ jQuery ”
链式操作
隐式迭代
3 DOM对象和jQuery对象
DOM对象和jQuery对象分别拥有一套独立的方法，不能混用
$(DOM对象) DOM对象转jQuery对象
jQuery对象[0] 或者 jQuery对象.get(0) jQuery对象转DOM对象

六、jQuery选择器

1 基本选择器：
id选择器: $(“#title”)
类选择器: $(“.title”)
标签选择器:$(“p”)
并集选择器：$(“#title, .title,p”)
全局选择器：$(“*”)
2 层次选择器：
后代选择器：$(“p span”)
子选择器： $(“p>span”)
相邻选择器：$(“p+span”)
同辈选择器：$(“p~span”)
3 属性选择器：$(“[name=pwd]”)
4 过滤选择器：
基本过滤选择器：
$(“p:first”) 第一个 $(“li:last”) 最后一个
$(“p:even”) 偶数 $(“li:odd”) 奇数
$(“p:eq(i)”) 第i个（从0开始）
$(“p:focus”) 获得焦点的
$(“p :header”)标题元素（h1-h6）
可见性过滤选择器：$(“p:hidden”) 隐藏的 $(“p:visible”) 可见的

七、jQuery中的事件与动画

1 jQuery中的事件
基础事件
鼠标事件 click( )，mouseover( )，mouseout( )
键盘事件 keydown( )，keyup( )，keypress( )
window事件 ready()载入事件
表单事件 submit()表单提交 blur() 失去焦点 focus() 获取焦点 select() 突出显示选择的文本区域
复合事件
鼠标光标悬停 hover( ) 相当于mouseover与mouseout事件
鼠标连续点击 toggle( ) 模拟鼠标连续click事件
2 绑定事件与移除事件
bind(“事件名”,函数) 单个事件绑定
bind({“事件名”:fn1,”事件名”:fn2,”事件名”:fn3}) 多个事件绑定
unbind(“事件名”,函数名)
unbind() 移除所绑定的全部事件
3 jQuery动画效果
控制元素显示与隐藏 show(时间) 显示 ，hide(时间 ) 隐藏
改变元素的透明度 fadeIn(时间) 淡入，fadeOut() 淡出
改变元素高度 slideDown() 延伸，slideUp() 缩短
自定义动画 $(选择的元素). animate({动画的CSS属性},时间,完成后执行的函数)

八、使用jQuery操作 DOM

1 操作CSS样式
css(“属性名”,”值”) 单个css设置
css({“属性名”:”值”,”属性名”:”值”,”属性名”:”值”}) 多个css属性设置
addClass(class); 追加样式
addClass(class1 class2 … classN);
removeClass(“class”); 移除样式
removeClass(“class1 class2 … classN “) ;
toggleClass() 模拟了addClass()与removeClass()实现样式切换的过程
hasClass( ) 判断是否包含指定的样式

2 操作文本与属性值内容
HTML代码操作
读 html();
写 html(“

…
“);
标签内容操作
读 text();
写 text(“
…
“);
元素的value属性值
读 val();
写 val(value);

3 操作DOM节点
append(content) 追加为最后的子节点
prepend(content) 插入到最前的子节点
after(content) 当前节点的后面 是兄弟
before(content) 当前节点的前面 是兄弟
remove()：删除整个节点
empty()：清空节点内容
replaceWith() 替换某个节点
clone()用于复制某个节点
4 属性操作
attr()用来获取与设置元素属性
removeAttr()用来删除元素的属性
5 遍历DOM节点
children() 获取元素的所有子元素
next() 紧邻匹配元素之后的元素
prev() 元素之前的元素
slibings() 所有同辈元素
parent()：获取元素的父级元素
parents()：元素元素的祖先元素
each( ) ：规定为每个匹配元素规定运行的函数
end( )：结束当前链条中的最近的筛选操作，并将匹配元素集还原为之前的状态

6 操作CSS-DOM
height() 设置或返回匹配元素的高度
width() 设置或返回匹配元素的宽度
offset() 返回以像素为单位的top和left坐标。仅对可见元素有效
scrollLeft() 设置或返回匹配元素相对滚动条左侧的偏移
scrollTop() 设置或返回匹配元素相对滚动条顶

九、表单校验

1 表单选择器 :input :text :password :radio :checkbox :hidden
2 表单属性过滤 :enabled :disabled :checked (匹配复选，单选和下拉列表) :selected
3 常用的正则表达式
/用户名验证/ /^[a-z][a-z0-9]{3,15}$/i; 名字要求是字母开头，总体是4-16位之间的数字和字母

/*密码验证*/        /^[a-z0-9]{4,10}$/i;   密码要求是4-10位的数字和字母

/*验证邮箱*/         /^\w{1,}@\w{1,}(\.\w{1,}){1,}$/;  

/*验证手机号码*/    /^1[0-9]{10}$/;

/*生日验证*/          /^((19[0-9]{2})|(200[0-9]))-(0{0,1}[1-9]|1[0-2])-(0{0,1}[1-9]|[1-2][0-9]|3[0-1])$/;

/*日期验证 yyyy-mm-dd */  /^\d{4}-(0[1-9]|1[0-2])-(0[1-9]|[1-2][0-9]|3[0,1])$/;
//简单的日期验证    /^\d{4}\-\d{2}\-\d{2}$/;

4.html5新增属性
placeholder 提供输入区域显示内容
required 规定输入域不能为空
pattern 规定验证输入与正则是否匹配

5.validity属性的validityState对象的属性
valueMissing 必填项 要求表单设置了required特性
typeMismatch 输入值是否和type匹配
patternMismatch 输入值是否和正则匹配
