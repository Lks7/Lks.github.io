    Git使用

    Git是分布式版本控制系统

    Git的基本命令

    1. 创建版本库
      git init //该命令使当前文件夹交给git管理

    2. 创建文件

      1. git add 文件名 //提交文件
      2. git commit -m “” //提交全部
      3. git status //查看当前仓库状态
      4. git diff //查看修改过的内容

    3. 查看历史日志以及回退版本

      1. git log //查看修改日志
      2. git reset -hard HEAD^ //表示回退到当前版本的上个版本，^^可以多写，太多可以写成HEAD~100
      3. git reflog //查看每一次使用的git命令

    4. git的分支

      1. 创建git分支

      2. 小结
        Git鼓励大量使用分支：

        查看分支：git branch

        创建分支：git branch

        切换分支：git checkout

        创建+切换分支：git checkout -b

        合并某分支到当前分支：git merge

        删除分支：git branch -d

      3. 如果合并分支时出现冲突，可以手动修改成想要的内容，并且解决冲突
        git log –grapg //可以查看分支合并图

    操作标签

    git tag -d v0.1 //该命令删除标签

    git push origin v1.0 //该命令将标签推送到远程

    git push origin –tags //一次性推送全部尚未推送到远程的本地标签

    小结

    配置git命令的别名

    如：git config –global alias.st status
    之后 我们就可以使用别名 git st来达到git status的效果
