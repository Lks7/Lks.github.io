  • TCP/IP协议

一.网络基础概念

  • 什么是计算机网络

    多个计算机之间的联系就叫计算机网络
  • 计算机网络的主要功能
    • 资源共享
    • 信息传输与集中处理
    • 均衡负荷与分布处理
    • 综合信息服务（www/综合业务数字网络 ISDN）

二.网络通信协议及接口

  • 什么是网络通信协议

    计算机网络中实现通信必须有一些约定即通信协议，对速率、传输代码、代码结构、传输控制步骤、出错控制等制定标准
    说白了就是它定义了一种语言，就像我们汉字一样，懂得了汉字就可以在本国内和其他人进行交流，网络也是
  • 网络通信接口
  • 为了使两个个结点之间能够进行对话，必须在它们之间建立通信工具（即接口），使彼此之间能进行信息交换。接口包括俩部分：
    • 硬件装置：实现结点之间的信息传送（网卡）
      • 软件装置：规定双方进行通信的约定协议 （通过网卡如何进行交流）

三.通信协议分层的思想

  • 分层：从接口、服务、协议三个层面来理解分层思想。
  • 接口：提供方法
  • 协议：具体实现
  • 服务：等于功能

分层思想十分广泛,不只应用在网络中

四.IP协议

这里推荐一本《TCP/IP详解》这本书详细的讲解了IP底层的实现

IP所做的最大的贡献就是ip地址，使每一台机器都有独一无二的ip地址，这使得电脑之间的通信不会混淆。

机器中使用4个字节来表示Ipv4，ip又分为公网ip和内网ip，内网ip又称其实就是局域网。

  • 公网
    • 真实的ip地址，占Ipv4所规定的范围内的一个ip。
  • 内网
    • 通过网关去访问外网，称为内网ip，这些内网中的电脑通过网关连接出去，都只是网关机器的ip地址。
子网掩码

子网掩码不能单独存在，它必须结合IP地址一起使用。子网掩码只有一个作用，就是将某个IP地址划分成网络地址和主机地址两部分。

如果我们想在内网中再分子网，这时使用ip提供的子网掩码，通过子网掩码可以在内网中在搭建子网，以此类推。

五.TCP协议和UDP协议

  • TCP(transmission control protocol)
    • 是专门设计用于在不可靠的因特网上提供可靠的、端到端的字节流通信的协议。它是一种面向连接的协议。TCP连接是字节流而非报文流，是一种可靠的连接。
      • 可靠的连接就是：我与你进行通信，必须要得到你的应答，只有你回应了我，我才能继续与你进行交流。就好比打电话，我们先说：喂，对方回应了这句喂之后在进行交谈，而且每说一句都得有回应，没了回应那么这个连接就断了。
  • UDP（user data protocol）
    • UDP向应用程序提供了一种发送疯转的原始ip数据报的方法、并且发送时无需建立连接。是一种不可靠的连接。
      • 不可靠连接：不管你有没有回应，我都将我说的话或者东西发给你，对方有没有收到，和我没有关系，反正会话已经发出。

TCP虽然可靠但是效率低，UDP不可靠但是快

一般玩游戏，视频通话是UDP，登录账号或者游戏平台为TCP。

六.Socket

一般的网络编程都叫Socket编程，Socket应用非常广泛，可以在不同的操作系统使用。

  • 两个JAVA应用程序可通过一个双向的网络通信连接实现数据交换，这个双向链路的一端称为一个Socket
  • Socket通常用来实现client-server连接
  • java.net包中定义的两个类Socket和ServerSocket，分别用来实现双向连接的client和Server端
  • 建立连接时所需的寻址信息为远程计算机的IP地址和端口号（Port number）
端口号

 所谓的端口，就好像是门牌号一样，客户端可以通过ip地址找到对应的服务器端，但是服务器端是有很多端口的，每个应用程序对应一个端口号，通过类似门牌号的端口号，客户端才能真正的访问到该服务器。

 在网络技术中，端口包括逻辑端口和物理端口两种类型，比如用于浏览网页服务的80端口，用于FTP服务的21端口等。如TCP/IP协议中的服务端口，通过不同的逻辑端口来区分不同的服务。一个IP地址的端口通过16bit进行编号，最多可以有65536个端口。端口是通过端口号来标记的，端口号只有整数，范围是从0 到65535。

强调一点：我们自己写程序时，最好不要写1024以下的端口号，因为1024以下的端口号随时可能被系统征用

1024-65535为用户端口,又分为:

BSD临时端口(1024-5000)和BSD服务器(非特权)端口(5001-65535)

0-1023: BSD保留端口,也叫系统端口,这些端口只有系统特许的进程才能使用;

1024-5000: BSD临时端口,一般的应用程序使用1024到4999来进行通讯;

5001-65535: BSD服务器(非特权)端口,用来给用户自定义端口.

端口又分为TCP端口和UDP端口,他们各自都拥有65536个端口号。
