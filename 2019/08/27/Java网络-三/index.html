总结

总结一下这三章所学的内容：

  1. 网络协议分层思想。
  2. IP的概念。
  3. TCP/UDP的概念。
  4. TCP/UDP程序的写法。

TCP/UDP的固定写法：

TCPServer端：

  1. 首先new一个ServerSocket监听在某一个端口。
  2. 调用accept()方法接收一个连接。
  3. 拿到连接就开始进行通话。
  4. 关闭掉连接。

TCPClient端：

  1. 首先new一个Socket连接到服务器上。
  2. 使用InputStream或者OutputStream进行通话。
  3. 最后关闭连接。

UDPServ端：

  1. new 一个DatagramSocket。
  2. new一个字节数组在new 一个DatagramPacket。
  3. DatagramPacket接收数据放在字节数组中。
  4. 通过DatagramSocket的receive()方法接收。

UDPClient端：

  1. 首先new一个ByteArrayOutputStream，转成字节数组。
  2. 在new一个DataOutputStream，将ByteArrayOutputStream装入这个包中。
  3. new一个DatagramSocket，使用DatagramSocket的send()方法将包发送出去。
